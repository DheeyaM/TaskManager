Task Manager

A modern task management web app built with React and Tailwind CSS to help users organize tasks efficiently and stay productive.

Features
Add and delete tasks
Mark tasks as completed
Clean and responsive UI
Fast and interactive experience
Built with reusable React components

React
Tailwind CSS
JavaScript
